## 项目简介
用于自动查找《鸣潮》游戏的抽卡链接，方便快速获取抽卡记录。

## 目录结构
- `main.py`：主程序文件
- `鸣潮抽卡记录.json`：抽卡记录数据文件，存储抽卡历史。
- `url.ps1`：PowerShell脚本，用于自动查找并复制鸣潮抽卡链接到剪贴板。

## 使用说明

### 1. 查找抽卡链接
- 双击或右键以 PowerShell 方式运行 `url.ps1`。
- 脚本会自动查找游戏日志中的抽卡链接，并复制到剪贴板。但必须最近在游戏中有查看过抽卡记录才行。
- 若未找到链接，就手工吧。

### 2. 抽卡记录获取
- 用 `pip install -r requirements.txt` 安装依赖并运行主程序。
- 填入抽卡链接并回车，获取成功后会生成记录文件。

## 环境要求
- Windows
- 具有 Python环境

## 注意事项
- 抽卡链接只有5分钟左右的有效期。
- 鸣潮只提供近半年的记录获取。
- 本项目是为方便个人使用。
